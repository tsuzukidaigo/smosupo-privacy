スモサポ プライバシーポリシー

スモサポは、アカウント登録なしで使える禁煙・減煙のセルフケアアプリです。改善データの共有は任意で、同意しなくても基本機能を利用できます。

最終更新日：2026年7月14日

1この端末だけに保存する記録

詳しい記録はこの端末だけ／iCloud保存・同期なし

開始日時、喫煙本数、ラプス、吸いたい波、ごほうび、メモ、写真、支援設定など、本人向けの詳しい記録はこの端末内だけに保存します。CloudKitなどを使ったiCloud保存・端末間同期は行わず、iCloudバックアップから除外する属性を保存領域へ設定します。このため、別の端末、再インストール後、機種変更後には詳しい記録を引き継げません。

これらの詳しい記録を、開発者の分析基盤へそのまま送ることはありません。

2同意した場合だけ共有する改善データ

任意のオプトイン／同意版 2026-07-14-v4

アプリ内の説明へ明示的に同意した場合だけ、次の情報をコード化して共有します。v4では、共有項目に加えて、処理委託先がCloudflareとSupabaseであること、および両社のプラットフォームログでリクエストのヘッダー・本文とIP関連情報が処理・一時保存される場合があることを同意画面へ明示しています。v3以前の許可をこの版へ引き継がず、改めて選択をお願いします。

改善データ送信先が未設定のビルドでは、同意しても外部送信は始まりません。同意の選択は端末に保存され、同じ同意版のまま送信先を有効にした更新後は送信が始まります。

  • 機能の利用：オンボーディング、利用日、ラプス記録、ラプス後の再訪・復帰確認、SOS、通知などの固定イベント
  • 支援プロフィール：1日の喫煙本数帯（1〜5、6〜10、11〜20、21〜40、41本以上）、起床後最初の1本までの時間帯（5分以内、6〜30分、31〜60分、61分以上、未回答）、禁煙・減煙の目的、注意したい場面の分類、支援の強さ
  • ラプス：本数帯（1本、2〜3本、4本以上）、きっかけの分類、朝・日中・夕方・深夜の時間帯
  • SOS：モード、利用経路、結果、所要時間帯
  • 購入フロー：継続提案の表示、購入開始、成功・キャンセル・保留・失敗の結果、固定された失敗分類
  • 技術診断：固定されたエラー分類、表示画面の分類、再試行できるかどうか
  • 振り返り・アンケート：「良くなった・同じ・難しかった」や離脱理由など、あらかじめ用意した選択肢の固定コード（自由記述なし）
  • 実験への露出：実験を有効にした場合の固定された実験名・割当コード（現行リリースでは実験を無効にしています）
  • 分析用メタデータ：ランダムな分析用ID、ランダムなセッション・エピソードID、出来事のUTC日付（時刻なし）、利用開始からの相対日、アプリ版

本数や時間は、たとえば「11〜20本」「夕方」のような粗い区分へ端末内で変換してから送ります。喫煙に関係する自己申告情報を含むため、App StoreではHealth（健康）として申告します。HealthKitから情報を取得することはありません。

改善データには氏名などの直接識別子を入れませんが、同じランダムな分析用IDで最大180日間のイベントを結び付けます。このため「完全な匿名データ」ではなく仮名化された、利用者に関連付けられるデータとして扱い、App StoreでもHealth・製品の操作・デバイスID・購入履歴・その他の診断データ・その他のユーザーコンテンツ（固定選択式アンケート回答）を「Linked to User: Yes」と申告します。6分類とも追跡には使いません。Health・製品の操作・購入履歴・その他のユーザーコンテンツはAnalytics目的、デバイスIDとその他の診断データはAnalyticsに加えて削除証明・濫用防止・短期運用のApp Functionality目的で扱います。

通常の改善イベントは端末キューへ入れてから最低2時間待ち、さらに0〜6時間の幅を加えてまとめて送ります。キューへ入れた正確な時刻は端末内だけで使い、送信内容には含めません。

同意前、同意しない場合、同意を停止した後は、新しい改善データを送りません。広告識別子（IDFA）は使わず、他のアプリやWebサイトを横断して追跡しません。

3改善データとして送らない情報

  • 正確な喫煙本数、出来事の時分秒・正確な発生時刻、正確な金額
  • メモ、自由記述、写真、位置情報
  • 氏名、メールアドレス、電話番号、住所、連絡先
  • 診断、受診、服薬情報、HealthKitの生データ
  • クラッシュダンプ、スタックトレース、端末ログやエラー本文
  • transaction ID、レシート、クレジットカードなどの支払い情報

4利用目的と提供先

改善データは、スモサポ内部のプロダクト改善と介入効果の検証にだけ使います。広告、利用者の評価・保険・雇用判断、他社データとの照合、データの販売には使いません。第三者が独自の目的で利用できる形でも提供しません。

保存・処理にはSupabaseのTokyoリージョンを、通信保護と濫用防止のgateway/WAFにはCloudflare Workersを利用します。両社はスモサポの指示に基づく処理委託先であり、独自の広告目的には利用させません。アプリはCloudflare Workerの公開URLだけへ接続し、データベースやSupabase Edge Functionへ直接接続しません。

CloudflareとSupabaseは、通信の配信、セキュリティ、濫用防止、障害調査のため、IPアドレスやIP関連ヘッダー、受信時刻などのネットワーク情報を処理する場合があります。CloudflareはWorkerからSupabaseへの通信へIP関連ヘッダーを付ける場合があり、Supabase hosted Edge FunctionのInvocation logsはrequest/responseのheaders・bodyを記録するため、送信した固定コードのpayloadがプラットフォームログへ短期複製される場合があります。

スモサポのWorker/Edgeコードは本文、IP、分析用ID、削除用シークレットを独自のconsole logへ出さず、IPを分析テーブルへ保存しません。プラットフォームログを行動分析には使わず、log drainsで別サービスへ転送しません。本番で利用するCloudflare/Supabase planの実保持期間と削除可否を公開前に確認し、具体的な期間を本ポリシーへ反映します。確認が完了するまで改善データ送信を有効にしません。

5保持期間と運用メタデータ

  • 仮名化された分析用IDとコード化した生イベント：出来事のUTC日付と受信UTC日付の早い方を1日目として最大180 UTC暦日
  • 個別の分析用IDを持たない統計集計：24か月（10未満の異なる分析用IDしか含まない詳細な集計区分は保存しません）
  • 削除後の再作成を防ぐtombstone：削除日DからD+29までの30 UTC暦日。ランダムな分析用IDと削除日・拒否期限のUTC日付だけを保持し、D+30に失効します（イベント内容や削除用シークレットは保持しません）
  • 分析DB内の濫用防止rate-limitカウンター：10分以内
  • Cloudflare / Supabaseのプラットフォームログ：利用planの実保持期間（公開前に具体的日数を確認して追記し、未確認の間は改善データ送信を有効にしません）
  • バックアップ：30日以内に失効

gatewayやサーバーが技術的に記録する受信時刻とネットワーク情報は、障害対応と濫用防止のための運用メタデータです。受信時刻を喫煙・ラプス・SOSの発生時刻として扱わず、介入効果の分析にも使いません。上記の「10分以内」は分析DB内のrate-limit rowだけを指し、Cloudflare/Supabaseのプラットフォームログ保持とは別です。

6同意の停止と削除

設定画面から改善データの共有をいつでも停止できます。停止後も、この端末だけにある本人向け記録はそのまま利用できます。

「共有済みの改善データを削除」を実行すると、共有同意を停止し、未送信キューを消したうえで、分析用IDに結び付く生イベントの削除を依頼します。削除後の遅延再作成を防ぐため、ランダムな分析用IDと削除日・拒否期限のUTC日付だけのtombstoneを削除日DからD+29まで保持し、D+30に失効させます。tombstoneにはイベント内容や削除用シークレットを残しません。バックアップ内の複製は30日以内に失効します。

Cloudflare/Supabaseのプラットフォームログとバックアップは、アプリ内の削除操作で個別行を直ちに削除できない場合があります。これらには各サービスで確認した保持期限を適用し、期限後に失効させます。実保持期間と削除可否を確定して本ポリシーへ記載するまでは、改善データ送信を有効にしません。

端末内のラプス記録を後から編集したり1件だけ削除したりしても、すでに共有したコード化イベントは個別に訂正・削除されません。共有済みデータを消したい場合は、設定画面から分析用IDに結び付くデータ全体の削除を実行できます。

10以上の異なる分析用IDをまとめ、個別の分析用IDを持たない長期統計集計は、特定の分析用IDへ戻せないため分析用ID単位の削除対象外です。直近180日分の集計は、共有済みデータの削除後24時間以内に再構築して該当IDの寄与を除きます。180日を過ぎた集計から特定の分析用IDだけの寄与を取り除くことはできません。分析用IDは端末のインストール単位でランダムに作るため、この閾値は人数を保証するものではありません。

この端末の本人向け記録は、設定画面の「データを削除」から削除できます。アプリを削除した場合も端末内の記録は削除されますが、すでに共有した改善データは自動では消えません。共有済み改善データも消したい場合は、アプリを削除する前に設定画面から削除を実行してください。

7端末上の表示と安全管理

通知は端末内で作成し、ロック画面に詳しい喫煙記録を表示しない一般的な文面を使います。通知はiOSの設定から停止できます。ウィジェットは端末を見られる人に表示される可能性があるため、iOSの設定から取り外せます。

通信はHTTPSで暗号化します。分析用IDと削除用シークレットは端末のKeychainで保護し、削除用シークレットそのものをサーバーへ保存しません。管理者権限を最小限にし、通常の分析では10未満の異なる分析用IDしか含まない集計区分を表示・保存しません。

8課金（サブスクリプション）

最初の30日間はアプリ内で全機能を無料にします。30日後も続ける場合だけ、本人が購入操作を行って月額200円のサブスクリプションを開始します。App Store側の自動更新型無料トライアルは設定しません。SOSとラプス記録は未課金でも利用できます。

決済はAppleが処理し、開発者はクレジットカード情報を受け取りません。
